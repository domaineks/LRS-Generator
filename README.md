# Labor Remuneration Statement Generator

A web-based tool to quickly generate professional labor remuneration statements compliant with Taiwan tax law.

## Features

- **Automatic Tax Calculation**: Calculates withholding tax and supplementary health insurance based on Taiwan 2025 tax regulations
- **Multiple Income Types**: Supports salary (50), professional services (9A), royalties (9B), and other income (92)
- **ID Card Upload**: Upload front and back ID card images (optional)
- **Real-time Preview**: Instantly displays calculated amounts
- **PDF Export**: Generates professional payment statements
- **Privacy First**: 100% client-side processing, no data uploaded to servers

## Quick Start

1. Open `index.html` in your browser
2. Fill in the form (name, income type, amount, etc.)
3. Optional: Upload ID card images and payment details
4. Click "Download Payment Statement PDF"
